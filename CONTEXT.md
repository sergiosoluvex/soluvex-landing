## Servicios

### 01 — Webs Premium
Sitios web construidos a medida desde cero. Sin constructores, sin templates.
Cada web es un argumento visual diseñado para convencer antes de que el visitante lea una sola línea.

Incluye:
- Diseño editorial único por proyecto
- Animaciones GSAP + scroll cinematográfico
- SEO técnico y rendimiento optimizado
- Deploy completo en producción

Stack habitual: HTML + CSS + JS vanilla con GSAP + Lenis. Next.js para proyectos que requieran CMS o lógica de servidor.

### 02 — Automatizaciones IA
Flujos inteligentes que trabajan de forma autónoma.
El negocio respondiendo, cualificando y cerrando sin intervención manual.

Incluye:
- Agentes de WhatsApp y email
- Cualificación automática de leads
- Asistentes GPT a medida por sector
- Automatización de flujos internos
- Integración con CRM y herramientas existentes

## Proceso de trabajo
1. **Briefing** — conversación inicial para entender el negocio, objetivos y estándar esperado
2. **Propuesta** — propuesta estratégica y visual con alcance definido
3. **Desarrollo** — ejecución con actualizaciones directas al cliente
4. **Entrega** — deploy en producción, formación básica si aplica

Comunicación directa en cada fase. Sin gestores de cuenta, sin cadena de producción.
